class cpes.close_packings_via_layers.ClosePacking(num, radius, num_vector, *args, **kwargs)

Bases: Points3D

Represents a close-packed structure of spheres, extending the functionality of Points3D.

This class is designed to construct and manipulate close-packed structures, such as face-centered cubic (fcc) or hexagonal close-packed (hcp) systems, using layers of spheres.
